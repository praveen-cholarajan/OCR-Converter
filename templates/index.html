Installation Guide

Quick Install

We've created an automatic installer script to set up all dependencies correctly. 

python install_dependencies.py

Python Dependencies

Install required Python packages using pip:

pip install -r requirements.txt

This includes Flask, PyTesseract, python-docx, pdf2image, Pillow, and others.

Optional OCR Engines:

If you want to use EasyOCR or PyOCR, install their specific requirements:

pip install -r requirements-easyocr.txt
pip install -r requirements-pyocr.txt

Note: EasyOCR might require PyTorch. Follow PyTorch installation instructions if needed.

Poppler (PDF Utilities)

Required for converting PDF pages to images.

macOS
brew install poppler
Ubuntu/Debian
sudo apt-get update && sudo apt-get install poppler-utils
Windows
  1. Download the latest Poppler binary from Poppler for Windows Releases.
  2. Extract the archive (e.g., using 7-Zip).
  3. Copy the contents of the `bin/` directory from the extracted Poppler folder to a location included in your system's PATH environment variable (e.g., `C:\poppler\bin`).
  4. Alternatively, add the Poppler `bin/` directory directly to your system's PATH.
  5. You might need to restart your command prompt or system for the PATH changes to take effect.

Tesseract OCR

The primary OCR engine.

macOS
brew install tesseract

To install all language packs:

brew install tesseract-lang
Ubuntu/Debian
sudo apt-get update && sudo apt-get install tesseract-ocr

To install language packs (e.g., French):

sudo apt-get install tesseract-ocr-[lang]

Replace `[lang]` with the 3-letter language code (e.g., `fra`, `deu`, `spa`).

Windows
  1. Download the Tesseract installer from the Tesseract at UB Mannheim page (recommended).
  2. Run the installer. Make sure to select the language packs you need during installation.
  3. Ensure the Tesseract installation directory (e.g., `C:\Program Files\Tesseract-OCR`) is added to your system's PATH environment variable. The installer usually offers an option to do this.
  4. You might need to restart your command prompt or system for the PATH changes to take effect.

Troubleshooting

"Poppler not found" or PDF Conversion Errors
  • Ensure Poppler is installed correctly for your operating system.
  • Verify that the Poppler `bin` directory is in your system's PATH environment variable.
  • Restart the application or your terminal after modifying the PATH.
"Tesseract not found" or OCR Errors
  • Ensure Tesseract is installed correctly for your operating system.
  • Verify that the Tesseract installation directory is in your system's PATH environment variable.
  • Make sure you have installed the necessary language packs for Tesseract (e.g., `tesseract-ocr-eng`, `tesseract-ocr-fra`).
  • On Windows, you might need to explicitly tell PyTesseract where Tesseract is installed by setting `pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'` in `app.py` (adjust the path if necessary).
  • Restart the application or your terminal after modifying the PATH.
EasyOCR / PyOCR Errors
  • Ensure you have installed the specific requirements: `pip install -r requirements-easyocr.txt` or `pip install -r requirements-pyocr.txt`.
  • EasyOCR might require specific versions of PyTorch or CUDA. Check the EasyOCR documentation for compatibility.
  • PyOCR relies on an underlying OCR engine (usually Tesseract). Ensure Tesseract is installed and working.
